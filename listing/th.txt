NAME (33 chars, store limit 45):
Flobro - หน้าต่างลอยอยู่บนสุดเสมอ

SUMMARY (97 chars, limit 132):
ตรึงแท็บใดก็ได้ให้อยู่บนสุดเสมอในหน้าต่างลอย ปักหมุดวิดีโอ เอกสาร หรือแดชบอร์ด หรือลอยเหนือทุกแอป

DESCRIPTION:
ให้เว็บเพจใดก็ได้อยู่ในสายตาเสมอ Flobro เปลี่ยนแท็บ ลิงก์ หรือวิดีโอใดก็ได้ให้เป็นหน้าต่างลอยแบบมินิมอล: ไม่มีแท็บ ไม่มีแถบที่อยู่ ไม่มีอินเทอร์เฟซ มีแค่เนื้อหา มองเห็นได้ตลอดขณะคุณทำงาน

และ Flobro ไปไกลกว่าส่วนขยายอื่นทุกตัว ด้วยแอปเดสก์ท็อป Flobro ฟรีสำหรับ Mac และ Windows เพียงคลิกเดียว หน้าต่างลอยนั้นจะกลายเป็นหน้าต่างที่อยู่บนสุดเสมออย่างแท้จริง เหนือทุกแอปพลิเคชันบนหน้าจอ ไม่ใช่แค่ในเบราว์เซอร์

ฟีเจอร์ของ FLOBRO

- ลอยแท็บใดก็ได้: คลิกไอคอนบนแถบเครื่องมือ แท็บปัจจุบันจะเด้งออกมาเป็นหน้าต่างลอยไร้ขอบ
- ลอยลิงก์ใดก็ได้: คลิกขวาที่ลิงก์แล้วเปิดในหน้าต่างลอยได้ทันที
- ปักหมุดวิดีโอ ไลฟ์สตรีม แชท แดชบอร์ด หรือเอกสาร ให้อยู่ในสายตาตลอด
- ไร้สิ่งรบกวน: หน้าต่างแสดงเฉพาะเว็บเพจ ไม่มีอย่างอื่น
- ย้ายและปรับขนาดหน้าต่างลอยให้เข้ากับหน้าจอและการทำงานของคุณ
- อยู่บนสุดเสมอจริง ๆ: ส่งหน้าใดก็ได้ไปยังแอปเดสก์ท็อปฟรี แล้วหน้าต่างจะลอยเหนือทุกแอปพลิเคชัน พร้อมซูม (25-500%) จัดเรียง 16:9 และปักหมุด ในแถบเครื่องมือที่ปรากฏเฉพาะเมื่อเลื่อนเมาส์ไปวาง
- ฟรีและโอเพนซอร์ส (MIT) ไม่ต้องมีบัญชี ไม่มีโฆษณา

วิธีใช้งาน

1. คลิกไอคอน Flobro เพื่อลอยแท็บปัจจุบัน หรือคลิกขวาที่ลิงก์แล้วเลือก Flobro
2. ย้ายและปรับขนาดหน้าต่างลอยได้ตามใจ
3. อยากให้อยู่เหนือทุกแอป ไม่ใช่แค่ Chrome? ติดตั้งแอปเดสก์ท็อปฟรีจาก flobro.app แล้วส่งหน้าไปด้วยคลิกเดียว

ใครใช้ FLOBRO

- นักพัฒนา: เปิดเอกสาร บันทึกการ build หรือผลลัพธ์ API ไว้ดูขณะเขียนโค้ด
- นักเรียนนักศึกษา: ดูบทเรียนหรือบทช่วยสอนไปพร้อมจดโน้ตในแอปอื่น
- นักวิเคราะห์และเทรดเดอร์: ติดตามกราฟ ฟีด และข่าวสดโดยไม่ต้องสลับแท็บ
- นักเขียนและนักวิจัย: เปิดแหล่งอ้างอิงไว้ดูขณะเขียน
- แฟนกีฬาและวิดีโอ: ตามดูการแข่งขันหรือวิดีโอที่มุมหน้าจอ

คำถามที่พบบ่อย

ถาม: ทำอย่างไรให้แท็บ Chrome อยู่บนสุดเสมอ?
ตอบ: คลิกไอคอน Flobro เพื่อลอยแท็บในหน้าต่างไร้ขอบ หากต้องการหน้าต่างที่อยู่เหนือทุกแอปพลิเคชัน ส่งไปยังแอปเดสก์ท็อป Flobro ฟรีด้วยคลิกเดียว

ถาม: หน้าต่างลอยกับแอปเดสก์ท็อปต่างกันอย่างไร?
ตอบ: หน้าต่างลอยคือหน้าต่าง Chrome แบบมินิมอลไร้อินเทอร์เฟซ เบราว์เซอร์ไม่สามารถตรึงหน้าต่างไว้เหนือแอปอื่นได้ ดังนั้นเพื่อให้อยู่บนสุดเสมอจริง ๆ Flobro จะส่งหน้าให้แอปเดสก์ท็อปสำหรับ Mac และ Windows จัดการ

ถาม: Flobro ฟรีจริงไหม?
ตอบ: จริง Flobro ฟรีและโอเพนซอร์สภายใต้สัญญาอนุญาต MIT การพัฒนาได้รับการสนับสนุนจากเงินบริจาคโดยสมัครใจ

ถาม: แล้วความเป็นส่วนตัวของฉันล่ะ?
ตอบ: สถิติการใช้งานแบบไม่ระบุตัวตนจำกัดเฉพาะชื่อโฮสต์เท่านั้น ไม่มี URL เต็ม และปิดได้ในตัวเลือก รายละเอียดที่ flobro.app/privacy.html
