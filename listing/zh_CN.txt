NAME (16 chars, store limit 45):
Flobro - 窗口置顶悬浮窗

SUMMARY (46 chars, limit 132):
让任何标签页在悬浮窗中始终置顶。将视频、文档或仪表盘固定在无边框小窗，或悬浮在所有应用之上。

DESCRIPTION:
让任何网页始终保持在视线内。Flobro 可以把任何标签页、链接或视频弹出为极简悬浮窗：没有标签栏、没有地址栏、没有界面。只有内容本身，在你工作时始终可见。

Flobro 比其他任何扩展更进一步。配合免费的 Flobro 桌面应用（Mac 和 Windows），只需一键，悬浮窗就会变成真正始终置顶的窗口，凌驾于屏幕上所有应用之上，而不仅仅是浏览器。

FLOBRO 功能

- 悬浮任何标签页：点击工具栏图标，当前标签页立即弹出为无边框悬浮窗。
- 悬浮任何链接：右键点击链接，直接在悬浮窗中打开。
- 固定视频、直播、聊天、仪表盘或文档，让它们一直保持在画面上。
- 零干扰：小窗只显示网页本身，没有其他任何元素。
- 自由移动和调整悬浮窗大小，适配你的屏幕和工作流程。
- 真正始终置顶：将页面转发到免费的 Flobro 桌面应用，它就会悬浮在「所有」应用之上，并提供缩放（25-500%）、16:9 对齐和固定控制，工具栏仅在鼠标悬停时出现。
- 免费且开源（MIT）。无需账号，没有广告。

使用方法

1. 点击 Flobro 图标让当前标签页悬浮，或右键任意链接选择 Flobro。
2. 随意移动和调整悬浮小窗的位置和大小。
3. 想让它凌驾于所有应用之上，而不仅是 Chrome？从 flobro.app 安装免费桌面应用，一键转发页面。

谁在使用 FLOBRO

- 开发者：写代码时让文档、构建日志或 API 响应保持可见。
- 学生：一边看课程或教程，一边在其他应用做笔记。
- 分析师和交易者：实时盯盘、跟踪数据和新闻，无需切换标签页。
- 写作者和研究者：写作时让资料来源和参考文献一直在眼前。
- 体育和视频爱好者：在屏幕角落追比赛或视频。

常见问题

问：如何让 Chrome 标签页始终置顶？
答：点击 Flobro 图标，让标签页在无边框小窗中悬浮。若要让窗口凌驾于所有应用之上，只需一键将它转发到免费的 Flobro 桌面应用。

问：悬浮小窗和桌面应用有什么区别？
答：小窗是没有界面的极简 Chrome 窗口。浏览器无法让窗口保持在其他应用之上，因此要实现真正的始终置顶，Flobro 会把页面交给 Mac 和 Windows 桌面应用处理。

问：Flobro 真的免费吗？
答：是的。Flobro 免费且以 MIT 许可开源，开发由自愿捐赠支持。

问：我的隐私呢？
答：匿名使用统计仅限主机名，绝不包含完整网址，并可在选项中关闭。详情请见 flobro.app/privacy.html。
