NAME (17 chars, store limit 45):
Flobro - 永遠置頂懸浮視窗

SUMMARY (48 chars, limit 132):
讓任何分頁在懸浮視窗中永遠置頂。將影片、文件或儀表板釘選在無邊框小窗，或懸浮在所有應用程式之上。

DESCRIPTION:
讓任何網頁永遠保持在視線內。Flobro 可以把任何分頁、連結或影片彈出成極簡懸浮視窗：沒有分頁列、沒有網址列、沒有介面。只有內容本身，在你工作時始終可見。

Flobro 比其他任何擴充功能更進一步。搭配免費的 Flobro 桌面應用程式（Mac 與 Windows），只要一鍵，懸浮視窗就會變成真正永遠置頂的視窗，凌駕於螢幕上所有應用程式之上，而不只是瀏覽器。

FLOBRO 功能

- 懸浮任何分頁：點擊工具列圖示，目前的分頁立即彈出成無邊框懸浮視窗。
- 懸浮任何連結：在連結上按右鍵，直接在懸浮視窗中開啟。
- 釘選影片、直播、聊天室、儀表板或文件，讓它們一直保持在畫面上。
- 零干擾：小窗只顯示網頁本身，沒有其他任何元素。
- 自由移動與調整懸浮視窗大小，配合你的螢幕與工作流程。
- 真正永遠置頂：將頁面轉送到免費的 Flobro 桌面應用程式，它就會懸浮在「所有」應用程式之上，並提供縮放（25-500%）、16:9 對齊與釘選控制，工具列只在滑鼠移入時出現。
- 免費且開放原始碼（MIT）。無需帳號，沒有廣告。

使用方式

1. 點擊 Flobro 圖示讓目前分頁懸浮，或在任何連結上按右鍵選擇 Flobro。
2. 隨意移動與調整懸浮小窗的位置和大小。
3. 想讓它凌駕於所有應用程式之上，而不只是 Chrome？從 flobro.app 安裝免費桌面應用程式，一鍵轉送頁面。

誰在使用 FLOBRO

- 開發者：寫程式時讓文件、建置紀錄或 API 回應保持可見。
- 學生：一邊看課程或教學影片，一邊在其他應用程式做筆記。
- 分析師與投資人：即時盯盤、追蹤數據與新聞，不必切換分頁。
- 作家與研究者：寫作時讓資料來源與參考文獻一直在眼前。
- 體育與影音愛好者：在螢幕角落追賽事或影片。

常見問題

問：如何讓 Chrome 分頁永遠置頂？
答：點擊 Flobro 圖示，讓分頁在無邊框小窗中懸浮。若要讓視窗凌駕於所有應用程式之上，只要一鍵將它轉送到免費的 Flobro 桌面應用程式。

問：懸浮小窗和桌面應用程式有什麼差別？
答：小窗是沒有介面的極簡 Chrome 視窗。瀏覽器無法讓視窗保持在其他應用程式之上，因此要達到真正的永遠置頂，Flobro 會把頁面交給 Mac 與 Windows 桌面應用程式處理。

問：Flobro 真的免費嗎？
答：是的。Flobro 免費且以 MIT 授權開放原始碼，開發由自願捐款支持。

問：我的隱私呢？
答：匿名使用統計僅限主機名稱，絕不包含完整網址，並可在選項中關閉。詳情請見 flobro.app/privacy.html。
